# FED Chatbot

A standalone chatbot UI for the Federation of Entrepreneurship Development (FED) website.

## Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm run dev
# Opens at http://localhost:5173

# Build for production
npm run build

# Preview production build
npm run preview
```
